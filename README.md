# Gestor de Tareas en Java

Aplicación de consola desarrollada en Java como proyecto de práctica para el ciclo
de Desarrollo de Aplicaciones Multiplataforma (DAM).

## Funcionalidades
- Añadir tareas
- Mostrar la lista de tareas
- Eliminar tareas
- Salir de la aplicación

## Tecnologías utilizadas
- Java
- ArrayList
- Scanner
- Programación estructurada

## Descripción
La aplicación muestra un menú por consola que permite al usuario gestionar
una lista de tareas introducidas por teclado. Se utilizan estructuras básicas
de control, colecciones y entrada/salida estándar.

## Objetivo del proyecto
Afianzar los conocimientos básicos de Java y la lógica de programación
mediante un proyecto práctico.
